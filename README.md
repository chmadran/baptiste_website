# Getting Started 

Run `npm install`   
Run `npm run start`   
The following should open in your browser : `localhost:3000` 

# To push a change 

In the terminal, run :
* `git add --all` 
* `git commit -m "your change description"`
* `git push`

# To find a place to make a change : 

* `CTRL + SHIFT + F`
